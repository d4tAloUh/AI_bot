# AI_bot

### Firstly<br>
Create file called .env in environment folder
```
TOKEN=token_of_telegram_bot
```

### Starting<br>
**To start up the project do the following commands**<br>
Requires python 3.6+
```
1. python -m venv myvenv
2. cd myvenv/Scripts/
3. activate
cd ../..
4. pip install -r requirements.txt
5. python main.py
```
